Starter Projects

Official Packages

Lenkrad web-starter

A clean and simple project setup ideal for web-projects and perfect to get started with LENKRAD.

  • Attribute routing enabled
  • Clean structure
  • Prepared for SQLite & MySQL
  • Github Template installation available
  • Additional CLI command & templating

Lenkrad REST-starter

A powerful REST-API starter setup. Ideal for your dedicated backend.

  • Attribute routing enabled
  • Clean structure
  • Route guards (authentication)
  • JWT stateless authentication & authorization
  • SQLite out of the box (one line to change to MySQL)
  • Video tutorial

Neoan.io-starter

A web project with complexity. This is ideal for the deep divers (and therefore not recommended as a first LENKRAD project)

  • Route guards (authentication)
  • Session + JWT based authentication
  • Glassmorphism design
  • Video tutorial
